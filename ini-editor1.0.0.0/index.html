使用帮助

INI配置编辑器使用指南

INI文件是一种简单的配置文件格式，由节（sections）和键值对（key-value pairs）组成。

基本操作

  • 新建文件：点击工具栏的"新建"按钮创建一个新的INI文件。
  • 打开文件：点击工具栏的"打开"按钮选择并打开一个已有的INI文件。
  • 保存文件：点击工具栏的"保存"按钮保存当前编辑的文件。
  • 另存为：点击工具栏的"另存为"按钮将当前文件保存为新文件。

编辑配置节

  • 添加配置节：点击左侧的"添加配置节"按钮，输入配置节名称。
  • 选择配置节：点击左侧列表中的配置节进行编辑。
  • 重命名配置节：选择配置节后，点击编辑器头部的"重命名"按钮。
  • 删除配置节：选择配置节后，点击编辑器头部的"删除"按钮。
  • 复制配置节：选择配置节后，点击编辑器头部的"复制"按钮。

编辑键值对

  • 添加键值对：在编辑器底部输入键名和值，点击"添加"按钮。
  • 编辑键值对：直接在输入框中修改键名或值。
  • 删除键值对：点击键值对右侧的"删除"按钮。
  • 移动键值对：点击键值对右侧的"上移"或"下移"按钮调整顺序。

高级功能

  • 代码视图：点击工具栏的"代码视图"按钮，以纯文本形式查看和编辑INI文件。
  • 验证：点击工具栏的"验证"按钮，检查INI文件格式是否正确。
  • 导出：点击工具栏的"导出"按钮，将配置导出为其他格式。
  • 导入：点击工具栏的"导入"按钮，从其他格式导入配置。

INI文件格式说明

  • 注释：以分号（;）或井号（#）开头的行为注释。
  • 配置节：用方括号（[]）包围的部分，如[section]。
  • 键值对：格式为key=value，位于配置节下方。
  • 特殊字符：值中包含空格或特殊字符时，建议使用引号包围。
