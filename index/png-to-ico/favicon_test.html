检查 Favicon 是否显示

请查看浏览器标签页左上角，确认图标是否正确显示。

Chrome

检查标签页左侧和地址栏左侧

Firefox

检查标签页左侧和地址栏左侧

Edge

检查标签页左侧和地址栏左侧

HTML 中引用 Favicon

在您的 HTML 页面的 <head> 部分添加以下代码：

<!-- 基本的ICO格式图标 -->
<link rel="icon" type="image/x-icon" href="/favicon.ico">

<!-- 可选：提供不同尺寸的PNG图标支持现代浏览器 -->
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">

<!-- 可选：为移动设备添加触摸图标 -->
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">

故障排除

如果图标没有显示：

  1. 确认 favicon.ico 文件已上传到网站根目录
  2. 清除浏览器缓存并刷新页面
  3. 尝试使用不同的浏览器查看
  4. 检查HTML中的链接路径是否正确
  5. 确认图标文件格式正确（标准ICO格式）
提示： 为了获得最佳兼容性，建议同时提供ICO格式和PNG格式的图标。 ICO格式兼容性最好，而PNG格式支持更多现代特性。
